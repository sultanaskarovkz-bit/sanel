СТРУКТУРА ПРОЕКТА
-----------------
  index.html       - главная (выбор услуги)
  elektro.html     - лендинг ЭЛЕКТРИКИ (для рекламы)
  santeh.html      - лендинг САНТЕХНИКИ (для рекламы)
  assets/
    css/style.css  - все стили
    js/main.js     - формы, маски, анимации
    images/        - папка для своих фотографий

КАК РАЗВЕРНУТЬ НА ХОСТИНГЕ
--------------------------
  1. Распакуйте zip в корень хостинга (или в нужную папку).
  2. По умолчанию точка входа - index.html.
  3. Для рекламы на электрику - запускайте на:
       https://ваш-домен.kz/elektro.html
  4. Для рекламы на сантехнику - запускайте на:
       https://ваш-домен.kz/santeh.html

  Если хостинг не любит .html в URL - переименуйте файлы
  в elektro/index.html и santeh/index.html, тогда ссылки
  будут /elektro/ и /santeh/ соответственно.

КУДА ЛЕТЯТ ЗАЯВКИ
-----------------
  По умолчанию форма открывает WhatsApp с готовым текстом.
  Чтобы заявки приходили в Telegram или на email - откройте
  файл assets/js/main.js и в начале (блок CONFIG) пропишите:

    whatsapp:          'ваш_номер_без_плюса'
    phone:             '+7 700 ХХХ ХХ ХХ'  (для отображения)
    telegramBotToken:  '1234567890:AAH...' (получить у @BotFather)
    telegramChatId:    'ваш_chat_id'       (узнать через @userinfobot)
    formsubmitEmail:   'master@вашсайт.kz' (email для заявок)

  Можно использовать только что-то одно или все сразу.
  Если ничего не задано - открывается WhatsApp.

ЗАМЕНА ФОТОГРАФИЙ
-----------------
  Сейчас в галерее подключены изображения с Unsplash (легально,
  лицензия позволяет коммерческое использование). Чтобы
  заменить на свои:

  1. Положите свои фото в /assets/images/, например:
       elektro-1.jpg, elektro-2.jpg, ...
       santeh-1.jpg, santeh-2.jpg, ...
  2. В elektro.html и santeh.html найдите блоки <img src="..."
     и поменяйте URL на относительный путь:
       <img src="assets/images/elektro-1.jpg">

  Рекомендуемый размер: 900x680 px, JPEG, до 200 КБ каждое.

КАК НАСТРОИТЬ ПОД БРЕНД КЛИЕНТА
-------------------------------
  Все основные цвета и переменные в assets/css/style.css
  в самом верху (блок :root и .theme-elektro / .theme-santeh).

  Логотип и название - в верстке (поиск "МАСТЕР НА ДОМ" и
  логотип-обертка <span class="logo-mark">M</span>).

  Номер телефона - один раз в assets/js/main.js в CONFIG.

ЧТО ВКЛЮЧЕНО НА КАЖДОЙ СТРАНИЦЕ
-------------------------------
  Шапка с навигацией и кнопкой вызова мастера
  Hero с формой заявки (выше первого экрана)
  Доверительные блок (бейджи, цифры)
  Боли клиента (давление на болевые точки)
  Полный список услуг с описаниями
  Процесс работы (4 шага)
  Галерея работ
  Отзывы клиентов
  FAQ (раскрывающиеся блоки)
  Финальная форма заявки с бонусом
  Подвал с контактами
  Плавающая кнопка вызова на всех экранах

SEO И АНАЛИТИКА
---------------
  Title и description прописаны для каждой страницы.
  Под Open Graph готовы (для красивого превью при шаринге).
  Иконки favicon встроены через data:URL.

  Чтобы добавить Яндекс.Метрику или Google Analytics -
  вставьте код в конец <body> на всех 3 страницах.

ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ
---------------------
  Никакого PHP, Node, баз данных. Чистый HTML/CSS/JS.
  Грузится на любом самом простом хостинге.
  Поддержка всех современных браузеров (Chrome, Safari,
  Firefox, Edge, мобильные).
  Адаптация под планшеты и телефоны включена.
