МАСТЕР НА ДОМ - Самодостаточная версия
================================================

Каждый файл (index.html, elektro.html, santeh.html) полностью самодостаточен.
CSS и JavaScript встроены прямо в HTML.

КАК ОТКРЫТЬ
-----------
Просто двойной клик по любому файлу - откроется в браузере.

КАК ЗАЛИТЬ НА ХОСТИНГ
---------------------
Загрузить три HTML файла в корневую папку сайта. Все.
Никакие папки assets/ больше не нужны.

КУДА БУДУТ ПРИХОДИТЬ ЗАЯВКИ
---------------------------
По умолчанию форма откроет WhatsApp с готовым текстом заявки.

Чтобы изменить - открой любой из трех HTML файлов в текстовом редакторе,
найди в самом низу блок CONFIG и замени значения:

const CONFIG = {
  whatsapp: '77001234567',        // твой WhatsApp без плюса
  phone: '+7 700 123 45 67',       // отображаемый номер
  telegramBotToken: '',            // токен Telegram бота
  telegramChatId: '',              // ID чата куда слать
  formsubmitEmail: ''              // email через formsubmit.co
};

Если хочешь Telegram + email + WhatsApp одновременно - заполни все поля.
Если оставишь только whatsapp - заявка пойдет в WhatsApp.

ВАЖНО: CONFIG нужно поменять в КАЖДОМ из трех файлов (или один раз,
а потом скопировать тот же блок в два других).

ЧТО ПОМЕНЯТЬ ПЕРЕД ЗАПУСКОМ
---------------------------
1. Название бренда МАСТЕР НА ДОМ - заменить на свое (поиск-замена)
2. Номер телефона +7 700 123 45 67 - заменить
3. Email info@master-astana.kz - заменить
4. Адрес ул. Достык, 13 - заменить
5. CONFIG в JS - настроить куда слать заявки
6. Фото с Unsplash - заменить на свои (по желанию, сейчас работает с плейсхолдерами)

ЛЕНДИНГИ ДЛЯ РЕКЛАМЫ
--------------------
- elektro.html - под рекламу электрики (желто-оранжевая тема)
- santeh.html - под рекламу сантехники (сине-бирюзовая тема)
- index.html - общая входная страница с выбором услуги

Каждый лендинг отправляет источник заявки в скрытом поле source,
так что в Telegram/email будет видно с какого лендинга пришел клиент.
